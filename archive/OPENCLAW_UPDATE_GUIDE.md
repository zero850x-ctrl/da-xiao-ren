## 🔧 故障排除

### 問題1: sudo需要密碼
```
解決: 輸入你的Mac登錄密碼
注意: 輸入時不會顯示字符，正常輸入即可
```

### 問題2: npm命令找不到
```
解決1: 檢查Node.js安裝
node --version
npm --version

解決2: 重新安裝Node.js
brew install node
```

### 問題3: 權限修復後仍然錯誤
```
解決: 檢查目錄權限
ls -la /Users/gordonlui/.npm
應該顯示: drwxr-xr-x   gordonlui  staff

如果不行，嘗試:
sudo chmod 755 /Users/gordonlui/.npm
```

### 問題4: 更新後版本不變
```
解決: 檢查安裝路徑
which openclaw
應該顯示: /usr/local/bin/openclaw

如果顯示其他路徑:
npm uninstall -g openclaw
npm cache clean --force
npm i -g openclaw@latest
```
