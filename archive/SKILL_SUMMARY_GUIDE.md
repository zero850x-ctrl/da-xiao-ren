# 📚 6個Skill學習總指南

## 📋 技能列表概覽

### 1. 🕸️ agent-browser (瀏覽器自動化)
- **狀態**: 可用 (agent-browser-clawdbot v0.1.0)
- **功能**: 無頭瀏覽器自動化、數據抓取、網頁操作
- **應用**: 加密貨幣數據收集、新聞抓取、市場監控

### 2. 📊 diagram-generator (圖表生成)
- **狀態**: 可用 (thought-to-excalidraw v1.0.0)
- **功能**: 思維導圖、流程圖、架構圖生成
- **應用**: 學習進度圖表、技術分析圖、系統架構圖

### 3. 🚢 ship-learn-next (學習系統)
- **狀態**: 替代方案 (continuous-learning v1.0.0)
- **功能**: 持續學習、知識管理、進度跟踪
- **應用**: 學習計劃管理、知識庫構建、進度監控

### 4. 📽️ pptx (PowerPoint操作)
- **狀態**: 可用 (pptx-creatord v1.0.0)
- **功能**: PowerPoint自動生成、模板管理、簡報創建
- **應用**: 學習報告簡報、交易總結、項目展示

### 5. 👤 personal-assistant (個人助理)
- **狀態**: 可用 (virtually-us v1.0.0)
- **功能**: 個人助理、日程管理、任務提醒
- **應用**: 學習提醒、交易提醒、日常任務管理

### 6. 🇨🇳 Humanizer-zh (中文人性化)
- **狀態**: 可用 (ai-humanizer v2.1.0)
- **功能**: 文本人性化、語氣優化、本地化處理
- **應用**: 中文報告優化、溝通語氣調整、內容本地化

   - 跟踪學習進度
   - 管理知識庫
   - 規劃學習路徑

6. **交互提醒** (virtually-us)
   - 發送學習提醒
   - 通知交易機會
   - 日常任務管理
