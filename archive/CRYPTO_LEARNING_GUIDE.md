## 🛡️ 風險管理規則

### **嚴格遵守：**
1. **2%止損** - 每筆交易最大損失不超過總資產2%
2. **倉位計算** - 根據止損距離計算倉位大小
3. **持倉限制** - 最多同時持有3個倉位
4. **時間管理** - 每30分鐘檢查一次持倉

### **風險計算公式：**
```
總資產 = $100,000
每筆最大風險 = 總資產 × 2% = $2,000

買入價 = $69,000
止損價 = $67,000
每單位風險 = $2,000

倉位大小 = 每筆最大風險 ÷ (買入價 - 止損價)
          = $2,000 ÷ $2,000 = 1.0 BTC
```
