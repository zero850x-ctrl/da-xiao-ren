## 測試案例

### 案例1: 斐波那契移動平均線計算

**問題**:
給定以下HSI價格數據，計算8日、13日、34日移動平均線，並分析趨勢：

價格序列: [18000, 18100, 18050, 18200, 18180, 18300, 18400, 18350, 18500, 18480, 18600, 18550, 18700, 18680, 18800, 18750, 18900, 18880, 19000, 18950]

**預期推理步驟**:
1. 計算8日移動平均線（最後8個價格的平均值）
2. 計算13日移動平均線（最後13個價格的平均值）
3. 計算34日移動平均線（如果數據足夠）
4. 分析均線排列：多頭排列（短期>中期>長期）或空頭排列
5. 判斷趨勢強度和方向

### 案例2: 平行通道識別

**問題**:
識別以下價格序列中的平行通道：

高點: [18500, 18600, 18550, 18700, 18680, 18800, 18750, 18900, 18880, 19000]
低點: [18000, 18100, 18050, 18200, 18180, 18300, 18400, 18350, 18500, 18480]
當前價格: 18800

**預期推理步驟**:
1. 識別價格高點並繪製上軌線（阻力線）
2. 識別價格低點並繪製下軌線（支撐線）
3. 檢查兩條線是否平行
4. 判斷當前價格在通道中的位置（上軌、中軌、下軌）
5. 分析突破或反彈的可能性

### 案例3: 黃金分割分析

**問題**:
HSI從18000上漲到19000，計算關鍵的斐波那契回撤水平：

擺動低點: 18000
擺動高點: 19000
斐波那契水平: [0.236, 0.382, 0.5, 0.618, 0.786]

**預期推理步驟**:
1. 計算價格波動範圍：19000 - 18000 = 1000
2. 計算各斐波那契回撤水平：
   - 0.236回撤: 19000 - 1000×0.236 = 18764
   - 0.382回撤: 19000 - 1000×0.382 = 18618
   - 0.5回撤: 19000 - 1000×0.5 = 18500
   - 0.618回撤: 19000 - 1000×0.618 = 18382
   - 0.786回撤: 19000 - 1000×0.786 = 18214
3. 分析各水平的支撐阻力意義
4. 判斷當前價格與這些水平的關係

### 案例4: RSI和成交量分析

**問題**:
分析以下價格和成交量數據的RSI和成交量關係：

價格: [18600, 18550, 18700, 18680, 18800, 18750, 18900, 18880, 19000, 18950]
成交量: [1200, 1150, 1300, 1250, 1400, 1350, 1500, 1450, 1600, 1550] (單位: 百萬股)

**預期推理步驟**:
1. 計算14日RSI值
2. 判斷RSI是否處於超買(>70)或超賣(<30)區域
3. 分析成交量與價格變化的關係：
   - 價漲量增：趨勢健康
   - 價漲量縮：趨勢可能反轉
   - 價跌量增：賣壓沉重
   - 價跌量縮：賣壓減輕
4. 綜合RSI和成交量判斷市場動能
