### 1. XGBoost安裝問題
**問題**: `XGBoost Library (libxgboost.dylib) could not be loaded`
**解決**: 
```bash
brew install libomp
export LDFLAGS="-L/opt/homebrew/opt/libomp/lib"
export CPPFLAGS="-I/opt/homebrew/opt/libomp/include"
pip install xgboost==2.1.4
```

### 2. 富途API股票代碼格式
**問題**: `Unknown stock 0005`
