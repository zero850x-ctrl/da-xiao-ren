### 🚀 peekaboo 功能概述：

#### **核心功能：**
1. **屏幕捕獲**：截圖、錄屏、UI元素識別
2. **交互控制**：鼠標點擊、鍵盤輸入、拖放
3. **應用程序管理**：啟動、關閉、窗口控制
4. **系統控制**：菜單、Dock、剪貼板

#### **常用命令示例：**
```bash
# 基本交互
peekaboo see --annotate --path /tmp/ui-map.png
peekaboo click --on "B1"
peekaboo type "文本" --return

# 應用程序控制
peekaboo app launch "Safari" --open "https://example.com"
peekaboo list windows --json

# 鍵盤鼠標
peekaboo move 500,300 --smooth
peekaboo hotkey --keys "cmd,space"
```
